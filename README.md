## Run it on your machine

1. Clone the project

   ```sh
   git clone https://github.com/karthik181194/Blogger-Diary.git
   ```

1. Install the dependencies and run commands for client server seperately

   ```sh
   npm install
   ```

3. Start the development server

   ```sh
   npm start
   ```
